# Style Packs

Style Packs are reusable production modules for visualizing Chloe Katastrophe in different creative contexts.

They do not define who Chloe is.

They define how Chloe is styled, lit, framed, and presented for a specific type of scene.

Canon lives in the Chloe archive.

Identity lives in Chloe Model v1.

Style Packs are interpretation layers.

PromptBuilder should combine:

Character Model
+ Performance Reference
+ Style Pack
+ Environment / Scene
+ Shot Direction
+ Negative Canon

Style Packs may be used by image, video, TikTok, album cover, and campaign generation workflows.
